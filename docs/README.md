# WillianShop
É um back-end escrito em python que usa FastApi para construção de uma API REST, a biblioteca JOSE para criação de token JWT, passlib para hashe de senha, SQLAlchemy como ORM, PostgreSQL como banco de dados, Docker e Docker Compose para criação e conteinerização do sisterma de um e-commercer. Escolhi desenvolver este projeto aplicar meus conhecimentos de arquitetura limpa e das referidas tecnologias.
